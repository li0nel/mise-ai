Implementation Specification

Screen Overview

The Timeline Journey variation presents the recipe builder as a vertical journey map. Each customization question is a "stop" on the timeline. The user progresses from top (analysis) to bottom (recipe ready), answering one question at a time. Completed stops collapse to a single line with an edit affordance. A connecting line tracks progress.

Interaction Flow

  • Each pill tap sends a plain-text chat message (e.g., "Chicken thigh"). The widget layer does not mutate state directly.
  • After a pill is selected, the backend returns an updated block with the next node set to active and the current node set to completed.
  • Tapping the edit icon on a completed node sends a message like "Change dietary needs." The backend re-expands that node in revision state and pauses the current active node.
  • The solid progress line height is computed by measuring the cumulative Y offset of completed nodes using onLayout.
  • On the URL flow fork, "Save & Cook Now" sends "Save and cook now" to the chat. "Customize This Recipe" sends "Customize this recipe" and the backend returns the full timeline with all question nodes.

Responsive Notes

  • Timeline gutter (40px left + 28px gap) is fixed. Content area takes remaining width.
  • On narrower devices (<360px), reduce gutter to 32px left and 20px gap.
  • Pills wrap naturally with flex-wrap. No horizontal scroll needed.
  • Expanded cards respect parent width. No fixed widths on cards.
  • The celebration card in Frame 5 is centered vertically when the timeline collapses.

What the Mock Does NOT Show

  • Transition animations between frames (node expanding/collapsing, line growing)
  • Haptic feedback on pill selection
  • Loading states between questions (server round-trip delay)
  • Scroll behavior when the timeline exceeds viewport (auto-scroll to active node)
  • Accessibility: screen reader announcements for progress, focus management
  • Error states (network failure, invalid response from backend)
  • The full timeline expansion animation when "Customize This Recipe" is tapped on the URL flow
  • Keyboard/voice input fallback if the user wants a choice not in the pills
